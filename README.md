# 동반이환 만성질환 계약자의 개인화 건강 개입 경로 생성 모형
## LLM 가드레일 기반 연령·성별 층화 분석을 통한 손해율 관리 연구

**Korean Journal of Insurance (보험학회지) 투고 논문 재현 코드**

---

## 개요

본 레포지토리는 고혈압·당뇨 동반이환 계약자를 대상으로 개인화된 건강 개입 경로를 생성하는 3단계 분석 모형의 구현 코드 및 실험 결과를 제공합니다.

국민건강영양조사(KNHANES) 2020–2024 데이터에 **연령군(중장년 40–59세 / 고령 60세+) × 성별(남/여) 교차 층화**를 적용하여 4개 독립 모형을 구축하고, 그룹별 맞춤형 반사실 개입 경로를 도출합니다.

> **관련 해외 논문**: Yang, M., & Chun, H. (2025). LLM-Guided Counterfactual Intervention for Comorbid Chronic Disease Risk in Health Insurance. *(해외 학술지 투고 예정)*  
> 본 레포지토리는 연령×성별 교차 층화 설계를 적용한 국내 보험학 연구의 독립 재수행 결과입니다.

---

## 분석 흐름

```
KNHANES 2020–2024
       │
       ▼
[1단계] XGBoost 다중분류 위험예측 모형
        연령군×성별 4개 독립 모형 (Optuna 최적화)
        중장년_남성 / 중장년_여성 / 고령_남성 / 고령_여성
       │
       ▼
[2단계] GPT-4o-mini 임상 가드레일 에이전트
        환자 프로파일 기반 동적 허용 범위 생성 (JSON)
        · 개입 충돌 방지 (나트륨↓ → 탄수화물 보상 차단)
        · 신체계측 일관성 강제 (BMI·허리둘레·체중 동방향)
        · 에너지 안전 하한 (현재값의 70%)
       │
       ▼
[3단계] DiCE 반사실 생성 + 단계적 폴백
        그룹별 4개 개입 경로 도출
        목표 달성 불가 시: 등급 0 → 1 → 2 순차 폴백
       │
       ▼
실험결과_연령성별층화.json
```

| 그룹 | 표본 수 | 동반이환 비율 | 모형 정확도 | 등급3 Recall | 달성 등급 |
|---|---|---|---|---|---|
| 중장년_남성 | 1,088 | 15.7% | 0.61 | 0.47 | 등급 0 (정상) |
| 중장년_여성 | 2,272 | 4.5% | 0.78 | 0.40 | 등급 0 (정상) |
| 고령_남성 | 1,324 | 39.7% | 0.39 | 0.56 | 등급 0 (정상) |
| 고령_여성 | 1,655 | 31.1% | 0.46 | 0.63 | 등급 1 (폴백) |

## 파일 구조

```
├── 동반이환 만성질환 위험 개입 경로 생성 모형.ipynb  # 전체 분석 코드
├── 실험결과_연령성별층화.json                        # 4개 그룹 반사실 결과
├── agent_config.pkl                                 # 모형 설정 및 피처 정보
├── df_final.pkl                                     # 전처리 완료 분석 데이터셋
├── model_중장년_남성.pkl                             # 중장년 남성 XGBoost 모형
├── model_중장년_여성.pkl                             # 중장년 여성 XGBoost 모형
├── model_고령_남성.pkl                               # 고령 남성 XGBoost 모형
├── model_고령_여성.pkl                               # 고령 여성 XGBoost 모형
└── knhanes_comorbid_2020_2024_age_stratified.csv    # 전처리 결과 CSV
```

> **원시 데이터(hn20~hn24_all.sas7bdat)**: 질병관리청 국민건강영양조사 홈페이지(https://knhanes.kdca.go.kr)에서 신청 후 이용 가능합니다. 개인정보 보호를 위해 본 레포지토리에는 포함되지 않습니다.

## 위험등급 정의

| 등급 | 정의 | 역할 |
|---|---|---|
| 등급 0 | 정상 (고혈압×, 당뇨×) | 최종 목표 |
| 등급 1 | 고혈압만 | 폴백 목표 1 |
| 등급 2 | 당뇨만 | 폴백 목표 2 |
| 등급 3 | 동반이환 (고혈압+당뇨) | 개입 대상 |
